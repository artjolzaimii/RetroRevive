# RetroRevive

### Classic Car Marketplace & Showroom Platform

RetroRevive is a full-stack web platform designed to connect classic car enthusiasts with rare and vintage vehicles.  
The platform allows users to explore classic cars available for purchase as well as book historically significant vehicles for showroom displays or events.

The goal of RetroRevive is to preserve classic automotive heritage while making it accessible through a structured digital platform.

---

# Project Overview

RetroRevive operates as a mediator between buyers, collectors, and event organizers.

The platform contains two main services:

### Buyable Cars Marketplace
Classic car owners can list vehicles for sale. Each listing includes specifications, price, and vehicle details allowing potential buyers to explore and discover rare vehicles.

### Showroom Rental System
Certain historic cars are not available for sale but can be rented for exhibitions, events, or private displays.

---

# Core Features

- Browse classic cars available for purchase
- Filter vehicles by brand and year
- View detailed car pages with specifications and images
- Book classic cars for showroom display
- User authentication system (registration and login)
- Personal user profiles with rental history
- Admin dashboard for managing platform data
- Rental statistics and analytics using charts

# System Architecture

The system consists of two main user roles.

### Customer
Users can:

- Browse classic cars
- Purchase listed vehicles
- Book showroom cars
- Manage personal profiles
- View rental history

### Administrator

Admins can:

- Manage cars
- Manage listings
- Manage rentals
- Manage users
- View platform statistics
